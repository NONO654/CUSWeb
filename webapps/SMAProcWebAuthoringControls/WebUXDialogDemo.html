WebUX Dialog Default Configuration

This is a default dialog with no buttons!


Content is set using content attribute.

  1. Dialog is created declaratively
  2. This dialog is positioned to top left using my-position and at-position attributes

This is a dialog with buttons!


Dialog content is inserted as the child of the web-component!

  1. Dialog is created declaratively
  2. Button to dialog are added using ok-button-flag, apply-button-flag, cancel-button-flag
  3. This dialog is positioned to bottom left using my-position and at-position attributes
  4. Maximized button has been added by adding maximize-button-flag attribute
