Weighted Sum Ranking Method

Weighted Sum is the best known and simplest method for evaluating the goodness of multiple alternatives in terms of several objectives. The score value of each data point is simply a sum of all objective values, each multiplied by the final weight factor calculated as described previously. For the Weighted Sum method to be valid all objectives must be normalized and brought into the same range, otherwise effects of different objective functions will not be equally represented.

The Overall Score values of each alternative are calculated as follows:

a.       All thresholds are evaluated; if any "Must Have" thresholds are violated for a given alternative, its Overall Score value is set to 0.0 independent of the objective values

b.      Individual parameter objective values are calculated as follows ("higher = better" formulation is used):

  • For minimization parameters, raw_obj_val = -1*parameter_val
  • For maximization parameters, raw_obj_val = parameter_val
  • For target parameters, raw_obj_val = -1*Abs(target - parameter_val)

c.       Raw objective values calculated as shown above for each objective parameter are scaled to be within [1...100] range - this is a scaled parameter objective value

d.      Scaled objective values are multiplied by the final weight factor calculated as described previously - this is a scaled and weighted parameter objective value

e.      For each data point, all scaled and weighted objective values are added up to get the weighted sum - this is the raw score for the data point

f.        Raw scores for all data points are scaled to be within [1...100] range - this is the Overall Score displayed in the table (recall that if the data point has any "Must Have" thresholds violated, its Overall Score value is set to 0.0). The Overall Score of each data point also becomes the score of the top level (root) parameter group.

g.       For each parameter group other than the top level (root) group, the score is calculated as the sum of scores of all sub-groups and parameters. No scaling is done for group scores - only the sum of all children scores is taken as the score.
