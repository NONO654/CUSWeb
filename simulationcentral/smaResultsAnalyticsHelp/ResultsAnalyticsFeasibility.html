Data points are graded based on specified requirements (thresholds, objectives, and priorities) and marked accordingly as:

 

  • Best Alternatives:

"Tradeoff" or "Pareto optimal" data points. No improvement can be made in any of the objectives without sacrificing at least one other objective.

  • Dominated:

Data points that satisfy all the thresholds, but for each dominated point another point with better objectives can be found among "Best Alternatives".

  • Infeasible:

Data points that violate one or more of the thresholds.

  • Excluded:

Data points that have been explicitly excluded from grading by the user. Rank and Score become zero and excluded data points are listed last in italics in table if displayed.

 

Deselecting Feasibility check boxes will filter the associated data points from data views. However, the data points will still be scored and ranked if not excluded.
